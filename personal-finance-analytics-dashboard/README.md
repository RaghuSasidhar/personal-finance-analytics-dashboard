# 💰 Personal Finance Analytics Dashboard

## 📊 Overview

This project analyzes **personal financial transactions** and converts raw data into an **interactive dashboard** using **Python 🐍 and Power BI 📈**.
It helps understand income, expenses, savings, and spending patterns.

---

## ⚙️ Workflow

Raw CSV Data 📄
⬇️
Python Data Processing 🐍
⬇️
Feature Engineering & Aggregation 📊
⬇️
Power BI Dashboard 📈

---

## 📂 Dataset

The dataset contains financial transactions with fields like:

* 📅 Date
* 📝 Description
* 🏷 Category
* 💵 Amount
* 🔁 Type (Income / Expense)
* 💳 Payment Mode

---

## 🧹 Data Processing

Using **Pandas**, the script performs:

* Cleaning missing or duplicate data
* Creating new features (Year, Month, Weekday)
* Separating **Income** and **Expense** columns
* Calculating **Savings and Savings %**
* Aggregating **monthly and category spending**

---

## 📁 Generated Data Files

| File                       | Purpose                           |
| -------------------------- | --------------------------------- |
| processed_transactions.csv | Clean transaction data            |
| monthly_summary.csv        | Monthly income, expense & savings |
| category_summary.csv       | Expense by category               |
| forecast_data.csv          | Predicted next month expense      |

## 📊 Dashboard Features

### 💳 KPI Cards

* Total Income
* Total Expense
* Total Savings
* Savings Rate
* Predicted Expense

### 📈 Income vs Expense Trend

Shows monthly financial trends.

### 📊 Expense by Category

Bar chart comparing spending categories.

### 🥧 Category Distribution

Pie chart showing percentage of spending.

### 🎛 Month Filter

Allows users to filter the dashboard by month.

---

## 🔮 Forecasting

A **Linear Regression model** predicts the **next month’s expected expense** based on historical data.
